## Description

This project implements the classic Tic-Tac-Toe game using the Dart programming language and Object-Oriented Programming (OOP).

The game is played in the console between two players:

- Player 1 (X)
- Player 2 (O)

The application validates all user input, prevents invalid moves, updates the board after every turn, detects winners (horizontal, vertical, and diagonal), and determines when the game ends in a draw.
